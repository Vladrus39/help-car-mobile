## 🚨 КОГДА ИСПОЛЬЗОВАТЬ "КОНФЛИКТ НА ДОРОГЕ":

### ✅ Подходящие ситуации:

1. **Агрессивное поведение:**
   - Другой водитель угрожает
   - Попытка блокировки автомобиля
   - Физическая конфронтация

2. **Дорожные конфликты:**
   - Спор после ДТП
   - Отказ составить протокол
   - Угрозы или шантаж

3. **Необходима защита:**
   - Вы чувствуете угрозу безопасности
   - Нужны свидетели/помощники
   - Ожидаете полицию и нужна поддержка

### ❌ НЕ используйте для:

- Обычных технических проблем (батарея, топливо)
- Простых ДТП без конфликта
- Мелких поломок

---

## 📞 КОГДА НАЖИМАТЬ "112":

### 🔴 КРИТИЧЕСКИЕ СИТУАЦИИ:

1. **Угроза жизни:**
   - Агрессия с оружием
   - Физическое насилие
   - Медицинская помощь срочно

2. **ДТП с пострадавшими:**
   - Есть раненые
   - Необходима скорая помощь
   - Пожар или утечка топлива

3. **Преступление:**
   - Ограбление
   - Кража автомобиля
   - Хулиганство

### ⚠️ ВАЖНО:

- **112** - единый номер для всех служб
- Звонок **БЕСПЛАТНЫЙ** с любого телефона
- Работает **БЕЗ SIM-карты**
- Работает даже при **заблокированном телефоне**

## 🔢 СПЕЦИАЛИЗИРОВАННЫЕ НОМЕРА:

### 👮 **102 - Полиция:**
```
Когда звонить:
- ДТП без пострадавших
- Угрозы и конфликты
- Административные нарушения
- Нужен наряд полиции
```

### 🚑 **103 - Скорая помощь:**
```
Когда звонить:
- Есть пострадавшие
- Медицинская помощь
- Плохое самочувствие
- Травмы любой тяжести
```

### 🔥 **104 - Пожарная служба:**
```
Когда звонить:
- Возгорание автомобиля
- Утечка топлива
- Пожар рядом
- Нужна специальная техника
```

## 🎬 СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ:

### Сценарий 1: Незначительное ДТП

```
Ситуация:
Легкое столкновение, никто не пострадал, 
но другой водитель агрессивный.

Действия:
1. ❌ НЕ нажимайте 112 (не критично)
2. ✅ Выберите "🚨 Конфликт на дороге"
3. ✅ Опишите ситуацию
4. ✅ Создайте запрос помощи
5. ✅ Если ситуация обостряется → 📞 102
```

### Сценарий 2: ДТП с угрозами

```
Ситуация:
ДТП, другой водитель кричит, угрожает,
есть легкие травмы.

Действия:
1. ✅ СРАЗУ нажмите "📞 112"
2. ✅ Сообщите о ДТП и угрозах
3. ✅ Затем создайте запрос "🚨 Конфликт"
4. ✅ Ожидайте помощь в безопасном месте
```

### Сценарий 3: Техническая проблема ночью

```
Ситуация:
Сломался автомобиль ночью в незнакомом месте,
чувствуете себя небезопасно.

Действия:
1. ✅ Выберите тип поломки (батарея/топливо)
2. ✅ Опишите, что вам некомфортно
3. ✅ Создайте запрос
4. ⚠️ Держите 112 наготове (на случай)
5. ✅ Оставайтесь в закрытой машине
```

## 💡 СОВЕТЫ ПО БЕЗОПАСНОСТИ:

### Перед созданием запроса:

1. **Оцените ситуацию:**
   - Есть ли реальная угроза?
   - Нужна ли полиция СЕЙЧАС?
   - Можно ли подождать помощь?

2. **Обезопасьте себя:**
   - Закройте двери машины
   - Включите аварийку
   - Оставайтесь в машине если небезопасно

3. **Звоните 112 без колебаний если:**
   - Угрожают жизни
   - Есть оружие
   - Физическое насилие
   - Медицинская помощь

### При использовании кнопки 112:

```
Что сказать диспетчеру:
✅ "Здравствуйте, нужна помощь"
✅ Ваше точное местоположение
✅ Что случилось (кратко)
✅ Есть ли пострадавшие
✅ Ваш номер телефона

Диспетчер спросит:
- Где вы находитесь?
- Что произошло?
- Кому нужна помощь?
- Ваши данные
```

Кнопка 112 - это просто **ссылка** на телефонный набор.
Приложение **НЕ знает** звонили ли вы.
Полная приватность гарантирована.
